How to run the User Registration & Login and User Management System With admin panel Project

Project features:

Login-Logout system
Select and give access of files to the user
Forget password mail send system

Project Description
 
In this project Admin allow his selected json files to a perticular user to give user his selected fiels access.
After this when user login to user panel he can see those allowed files by admin. And can aslo watch the json files in the Graphical representation.
In this project you are only allowed to select json files.

1. Download the  zip file

2. Extract the file and copy loginsystem folder

3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

4. Open PHPMyAdmin (http://localhost/phpmyadmin)

5. Create a database with name loginsystem

6. Import loginsystem.sql file(given inside the zip package in SQL file folder)

7.Run the script http://localhost/loginsystem (frontend)

8. For admin Panel http://localhost/loginsystem/admin
